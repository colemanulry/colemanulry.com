Mercury Package Management

The current package delivery process at Calvin University consists of thousand-lined spreadsheets and Google Forms. This is a rather flawed system for both the processor and package recipients. This app, as a solution, makes it easy to log packages, send recipients emails, and pick up packages. Learn more about it here.

Rate My Course

A spin-off of ratemyprofessors.com, this web app was designed to allow students to rate various courses within Calvin University's Computer Science department. For liberal arts students, this allows them to gain a greater knowledge of not only the professor of a particular course that ratemyprofessor.com would offer, but particular course ratings as well. Through this, they can go into each semester confident in the expectations of each course. Learn more about it here.

- Chat App -

I created this web app for my web development course. It's a simple chat app, where messages are stored in a Google firebase, and rendered onto the screen in order of the message's timestamp. You are able to choose your own name handle and color theme. Feel free to leave a message! See it deployed here.
